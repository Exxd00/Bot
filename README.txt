# GitHub Web Uploader (Flask UI)

✅ جاهز تمامًا للتشغيل!

## الملفات المضمنة:
- app.py → خادم Flask جاهز
- لا حاجة لأي تعديل
- يجب فقط ضبط متغير البيئة `GH_TOKEN`

## طريقة التشغيل:

1. تثبيت Flask:
```bash
pip install flask requests
```

2. تصدير المفتاح:
```bash
export GH_TOKEN=ghp_XXXX...
# أو على ويندوز:
# set GH_TOKEN=ghp_XXXX...
```

3. تشغيل الخادم:
```bash
python app.py
```

4. فتح المتصفح على:
```
http://localhost:5000
```

📂 لا توجد ملفات تحتاج للحذف.
✔️ كل شيء مهيأ للعمل مباشرة.
